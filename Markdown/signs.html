公式大括号

对于公式 f ( x ) = { 0 x = 0 1 x ≠ 0 f(x)=\begin{cases} 0& x=0\\ 1& x\neq 0 \end{cases} f(x)={01​x=0x̸​=0​ ，可以有以下三种写法：

$f(x)=\begin{cases}
    0& x=0\\
    1& x\neq 0
\end{cases}$

$f(x)=\left\{\begin{array}{ll}
    0& x=0\\
    1& x\neq 0
\end{array}\right.$

$f(x)=\left\{\begin{aligned}
    0& x=0\\
    1& x\neq 0
\end{aligned}\right.$
